Mooltipass Mini, l'outils pour les paranos

Le Mooltipass est un petit accessoire qui vous permet de stocker de manière externe et non connecté, vos différents mots de passes afin de réduire les risque de piratage.

Le mooltipass vous permettra de garder en sécurité vos mots de passes grâce à un système de stockage sécurisé.
Vos mots de passes seront stockés cryptés en AES-256 dans votre accessoire et sont, en plus, sécurisé par une carte et un code PIN.
Notez qu'au bout de 3 essais, l'outils se bloque définitivement.

Comment on l'utilise?

  • Branchez votre Mooltipass à votre ordinateur, tablette ou téléphone via USB (Le mooltipass n'a pas besoin de pilote)
  • Insérez la carte et entrez votre code PIN (Sans le PIN, la carte est inutile)
  • Visitez un site qui neccesite de se loguer (si vous utilisez le plugin, l'outil vous demandera alors si vous voulez utiliser le mot de passe enregistré, ou d'en sauvegarder un nouveau si c'est la première fois que vous vous loguez)
  • Si vous n'utilisez pas le plugin, vous pouvez demander au mooltipass d'entrer le mot de passe lui même

Le mooltipass est compatible sur tous les supports (Windows, Linux, Mac et également certains supports mobiles Apple et Android)
Il emule un clavier USB afin de pouvoir entrer les différents mots de passe lui même.

L'entrée des identifiants peut être simplifié grâce à l'utilisation du plugin disponible pour chrome, firefox et safari.

Quels sont les avantages?

Il permet de limiter les risques de piratage puisque l'outil entre le mot de passe pour vous.
C'est un outil non propriétaire donc tout le monde peut développer de nouvelles choses pour le Mooltipass.
Une plate-forme Open-source donc tout le monde peut vérifier la sécurité de l'outils.
Une plate-forme sécurisé car seul les codes vérifiés (testés par les constructeurs et vérifiés par communauté) sont implanté dans le Mooltipass.

Les codes sources sont tous disponibles pour études et vérifications par tous le monde. Vous pouvez accéder au contenu sur leur répertoire GitHub diponible cette adresse.

Le Mooltipass mini est prévu pour le mois de Mars et pour le prix de 80$ environ.
